# labirints-and-monster

Top-down браузерная игра: процедурный лабиринт, монстры с pathfinding'ом, twin-stick стрельба, fog of war, цветные двери, сундуки с power-up/debuff, поддержка Xbox-геймпада через Gamepad API.

## Запуск

```sh
python3 -m http.server 8080
# открыть http://localhost:8080
```

## Тесты

```sh
npm test
```

Использует встроенный `node --test` runner (Node 18+).

## Управление

| Действие | Клавиатура | Xbox-геймпад |
|---|---|---|
| Движение | WASD / стрелки | левый стик |
| Прицел | курсор мыши | правый стик |
| Стрельба | ЛКМ | правый стик отклонён (авто-fire) |
| Спринт | Shift | LT |
| Рывок (i-frames) | Space | A |
| Взаимодействие | E | X |

## Механики

- **Лабиринт** — генерируется процедурно (recursive backtracker), каждая игра новая.
- **Вход** — зелёный тайл, **выход** — жёлтый. Цель: дойти до выхода.
- **Fog of war** — видно только в радиусе ~5 тайлов вокруг игрока, исследованные клетки запоминаются приглушённо.
- **Монстры**: Wanderer (бродит, заметив — преследует), Chaser (всегда идёт к игроку), Guard (охраняет зону 3×3).
- **HP/броня/stamina** — 3 жизни, броня поглощает урон вместо HP, спринт тратит stamina.
- **Цветные двери и ключи** — 1–3 двери R/G/B на пути к выходу, ключи всегда достижимы.
- **Сундуки** — слепой выбор: 70% power-up (броня, аптечка, патроны, компас, приманка), 30% debuff (отравление, замедление, слепота).
- **Dash** — рывок 200 px с i-frames, проходит сквозь монстров.

## Стек

- Phaser 3 (через CDN)
- Vanilla ES-модули, без сборщика
- Pixel-art, сгенерированный программно через `Graphics.generateTexture`
