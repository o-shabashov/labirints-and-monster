# labirints-and-monster

Top-down браузерная игра: лабиринт, монстры, twin-stick стрельба, fog of war, поддержка Xbox-геймпада через Gamepad API.

## Статус

В разработке. Дизайн зафиксирован в [docs/superpowers/specs/2026-05-23-maze-monsters-design.md](docs/superpowers/specs/2026-05-23-maze-monsters-design.md).

## Стек

- Phaser 3 (через CDN)
- Vanilla ES-модули, без сборщика на старте
- Pixel-art, сгенерированный программно

## Запуск (будет позже)

```sh
python3 -m http.server 8080
# открыть http://localhost:8080
```
